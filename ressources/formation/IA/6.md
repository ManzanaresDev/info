# ¿Qué es NumPy?

NumPy significa:

Numerical Python

Su objetivo es trabajar con grandes cantidades de números de forma extremadamente rápida.

Una lista normal:

```python
numeros = [5,8,2,9]
```

Un array de NumPy:

```python
import numpy as np

numeros = np.array([5,8,2,9])
```

Visualmente parecen iguales.

Internamente son muy diferentes.

---

# ¿Por qué NumPy es tan rápido?

Las listas almacenan objetos de Python.

NumPy almacena datos directamente en memoria de forma continua.

```
Lista

[Objeto][Objeto][Objeto]

↓

Más lenta

----------------------

NumPy

[5][8][2][9]

↓

Mucho más rápido
```

# ¿Qué es Pandas?

Pandas trabaja sobre NumPy.

Está diseñado para manejar tablas de datos.

Imagina una hoja de Excel.

```
Nombre    Edad

Ana        21

Luis       34

Eva        19
```

Eso es exactamente un DataFrame.

---

# Crear un DataFrame

```python
import pandas as pd

datos = {

    "Nombre":["Ana","Luis","Eva"],

    "Edad":[21,34,19]

}

df = pd.DataFrame(datos)

print(df)
```

Resultado.

```
Nombre Edad

Ana     21

Luis    34

Eva     19
```

# Mini Proyecto

Supongamos un archivo:

```
estudiantes.csv
```

Contenido.

```
Nombre,Edad,Nota

Ana,20,17

Luis,19,15

Eva,22,19

Pedro,18,12
```

Programa.

```python
import pandas as pd

df = pd.read_csv("estudiantes.csv")

print(df.describe())

print(df["Nota"].mean())

print(df.sort_values("Nota",ascending=False))
```

Con muy poco código obtenemos un análisis completo.
