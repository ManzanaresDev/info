# Introducción

Cuando alguien escucha la palabra "matemáticas", suele pensar en ecuaciones complicadas y páginas llenas de símbolos.

En realidad, la IA utiliza las matemáticas como un idioma para describir datos y relaciones entre ellos.

Una red neuronal no "piensa". Calcula.

Cada decisión que toma es el resultado de miles o millones de operaciones matemáticas.

La buena noticia es que no necesitas ser matemático para entenderlas.

Solo necesitas comprender unos pocos conceptos fundamentales.

---

# ¿Por qué la IA necesita matemáticas?

Imagina que quieres que una IA diferencie perros y gatos.

El ordenador no entiende imágenes.

Lo único que entiende son números.

Por tanto, la imagen debe convertirse en una enorme tabla de números.

```
Imagen

↓

Números

↓

Modelo matemático

↓

Resultado
```

Las matemáticas permiten transformar el mundo real en información que un ordenador puede procesar.

---

# Los números

Todo comienza con los números.

Ejemplo:

```
Edad = 30

Altura = 175 cm

Peso = 72 kg
```

Para un ordenador, estos datos no tienen significado.

Son simplemente valores numéricos.

---

# Variables

Una variable es un nombre asociado a un valor.

Ejemplo en Python:

```python
edad = 30
altura = 175
peso = 72
```

Una IA utiliza millones de variables durante el entrenamiento.

---

# Coordenadas

Imagina un plano cartesiano.

```
Y

↑

5 |

4 |

3 |        ●

2 |

1 |

0 +--------------------→ X

   0 1 2 3 4 5
```

La posición del punto es:

```
(3,3)
```

Esto es un vector de dos dimensiones.

---

# Vectores

Un vector es simplemente una lista ordenada de números.

Ejemplo:

```
[5, 8]
```

Otro ejemplo:

```
[12, 18, 4]
```

En IA un vector puede representar:

- una fotografía
- una palabra
- una canción
- una persona
- un cliente

Todo termina convirtiéndose en vectores.

---

# ¿Qué representa un vector?

Supongamos un alumno.

```
Edad = 20

Altura = 180

Peso = 75
```

El ordenador lo almacena así:

```
[20,180,75]
```

Ese vector describe a la persona.

---

# Dimensiones

Cada número representa una característica.

```
[20,180,75]

 ↑  ↑   ↑

 │  │   │

 │  │   Peso

 │ Altura

 Edad
```

Decimos que tiene tres dimensiones.

Un modelo moderno puede trabajar con miles de dimensiones.

---

# Matrices

Una matriz es una colección de vectores.

Ejemplo:

```
Alumno A

[20,180,75]

Alumno B

[19,172,68]

Alumno C

[23,181,81]
```

Podemos escribirlo como:

```
[
 [20,180,75]

 [19,172,68]

 [23,181,81]
]
```

Esto es una matriz.

---

# ¿Por qué son importantes?

Las redes neuronales realizan millones de multiplicaciones entre matrices.

Por eso las GPU son tan rápidas.

Están diseñadas precisamente para multiplicar matrices.

---

# Ejemplo visual

```
Vector

[2,3]

↓

Matriz

[
 [2,3]

 [5,8]

 [1,7]
]
```

La mayoría de los datos utilizados por la IA terminan teniendo esta forma.

---

# Funciones

Una función transforma una entrada en una salida.

Ejemplo sencillo.

```
x

↓

Duplicar

↓

2x
```

Si:

```
x = 5
```

La salida será:

```
10
```

---

# Función matemática

```
f(x)=2x
```

Si:

```
f(8)
```

Resultado:

```
16
```

Una red neuronal está formada por miles de funciones conectadas entre sí.

---

# Gráfica de una función

```
Y

↑

8 |

6 |

4 |      /

2 |    /

0 +----------------→ X
```

La línea representa la relación entre entrada y salida.

---

# Probabilidad

La IA rara vez responde con un "sí" o un "no".

Normalmente calcula probabilidades.

Ejemplo.

```
Perro

97%

Gato

2%

Caballo

1%
```

La respuesta elegida será:

```
Perro
```

Porque tiene la mayor probabilidad.

---

# Estadística

La estadística permite descubrir patrones.

Ejemplo.

Notas:

```
7

8

5

9

6
```

Promedio:

```
7
```

Las IA utilizan estadísticas constantemente para analizar datos.

---

# Error

Recordemos el capítulo anterior.

La IA intenta minimizar el error.

Imagina una diana.

```
      X

   X

          X

      ●

```

Cada entrenamiento intenta acercar los disparos al centro.

---

# Derivadas

No veremos la parte complicada.

Solo la idea.

Una derivada responde a la pregunta:

> ¿En qué dirección debo moverme para reducir el error?

Es como bajar una montaña con niebla.

```
        /\

       /  \

      /    \

_____/______\____

      ↓

 Bajar
```

Las derivadas indican la pendiente.

---

# Gradiente

El gradiente combina muchas derivadas.

Indica cuál es la dirección que reduce más rápidamente el error.

Por eso el algoritmo Gradient Descent recibe ese nombre.

---

# Optimización

El proceso completo sería:

```
Modelo

↓

Predicción

↓

Error

↓

Derivadas

↓

Gradiente

↓

Actualizar parámetros

↓

Menor error
```

Después vuelve a comenzar.

Miles de veces.

Millones de veces.

Hasta aprender.

---

# Ejemplo sencillo en Python

```python
import numpy as np

# Vector

persona = np.array([20,180,75])

print(persona)

# Matriz

alumnos = np.array([
    [20,180,75],
    [19,172,68],
    [23,181,81]
])

print(alumnos)
```

Resultado:

```
[20 180 75]

[[20 180 75]
 [19 172 68]
 [23 181 81]]
```
