# Inspiración biológica

El cerebro humano contiene aproximadamente

```
86 000 000 000 neuronas
```

Cada neurona recibe señales de miles de neuronas vecinas.

Después decide si envía una nueva señal.

```
     Señales

        ↓

   [ Neurona ]

        ↓

 Nueva señal
```

Las redes neuronales se inspiran en esta idea.

No intentan copiar el cerebro.

Simplemente utilizan un modelo matemático parecido.

---

# La neurona artificial

Una neurona artificial recibe varios números.

Por ejemplo:

```
Altura

Peso

Edad
```

Representados como:

```
170

75

30
```

La neurona realiza un cálculo.

```
170

↓

75

↓

30

↓

Neurona

↓

Resultado
```

Ese resultado será enviado a la siguiente capa.

---

# Entradas

Las entradas suelen representarse mediante:

```
x₁

x₂

x₃

...

xn
```

Ejemplo.

```
Edad = 30

Altura = 175

Peso = 72
```

Vector:

```
x =

[30,175,72]
```

---

# Pesos

Aquí aparece el concepto más importante.

Cada entrada tiene un peso.

```
Edad ----------(0.2)

Altura --------(0.8)

Peso ----------(0.4)
```

¿Qué significa?

La red está indicando qué importancia tiene cada dato.

Si un peso vale:

```
0
```

Prácticamente ignora esa entrada.

Si vale:

```
5
```

La considera muy importante.

Los pesos son el verdadero conocimiento de una IA.

Entrenar un modelo significa encontrar los mejores pesos.

---

# El cálculo

Una neurona realiza:

```
entrada × peso

+

entrada × peso

+

entrada × peso

+

...

```

Matemáticamente.

```
x₁w₁+x₂w₂+x₃w₃
```

Ejemplo.

```
Edad

30 ×0.2

=

6

----------------

Altura

175×0.8

=

140

----------------

Peso

72×0.4

=

28.8
```

Suma.

```
6+140+28.8

=

174.8
```

Todavía no ha terminado.

---

# El sesgo (Bias)

Añadimos un valor adicional.

```
174.8

+

1.5

=

176.3
```

Ese número recibe el nombre de:

Bias

Permite desplazar la respuesta.

Sin él muchas redes serían incapaces de aprender correctamente.

La fórmula completa queda:

```
z = x·w+b
```

Donde:

```
x

↓

Entradas

w

↓

Pesos

b

↓

Bias
```

# Función de activación

Ahora tenemos un número.

Pero necesitamos decidir qué hacer con él.

Para eso usamos una función de activación.

```
z

↓

Función

↓

Salida
```

Sin funciones de activación...

Una red neuronal sería simplemente una gran ecuación lineal.

No podría resolver problemas complejos.

---

# La función escalón

Fue una de las primeras.

```
Si x>0

↓

1

Si no

↓

0
```

Gráficamente.

```
1 ─────────────

            │

            │

0────────────┘
```

Muy sencilla.

Pero poco útil para redes modernas.

---

# Sigmoid

Convierte cualquier número en un valor entre

```

y

1
```

```
        1

```

Gráfica aproximada.

```
1 |

  |        ______

  |      /

  |
```

Muy utilizada antiguamente.

# ReLU

```

```

```

```

Es rápida.

Simple.

Y funciona muy bien.

# Forward Propagation

Cuando introduces datos ocurre este proceso.

```
Imagen

↓

Primera capa

↓

Segunda capa

↓

Tercera capa

↓

Resultado
```

Eso recibe el nombre de

Forward Propagation.

Es simplemente avanzar desde la entrada hasta la salida.

---

# ¿Y si se equivoca?

Supongamos.

Respuesta correcta.

```
Perro
```

Respuesta del modelo.

```
Gato
```

Ha cometido un error.

Ahora debe aprender.

---

# Backpropagation

Este algoritmo revolucionó la IA.

Su misión consiste en responder.

```
¿Qué pesos debo modificar para reducir el error?
```

Proceso.

```
Salida

↓

Calcular error

↓

Retroceder

↓

Modificar pesos

↓

Nuevo intento
```

Observa que ahora viajamos en dirección contraria.

Por eso se llama

Backpropagation.

---

# Ciclo completo

```
Datos

↓

Forward

↓

Predicción

↓

Error

↓

Backpropagation

↓

Actualizar pesos

↓

Forward

↓

Nuevo error

↓

...

↓

Modelo entrenado
```

Este ciclo puede repetirse millones de veces.

---

# Ejemplo en NumPy

```python
import numpy as np

# Entradas
x = np.array([30, 175, 72])

# Pesos
w = np.array([0.2, 0.8, 0.4])

# Bias
b = 1.5

# Producto escalar
z = np.dot(x, w) + b

print(z)
```

Salida:

```
176.3
```

Acabas de implementar una neurona artificial.

# ¿Qué aprende realmente una red?

No aprende reglas.

No memoriza frases.

Aprende números.

Una red neuronal moderna puede contener:

```
1 millón

10 millones

100 millones

1 billón
```

de parámetros.

Cada parámetro es simplemente un número que cambia durante el entrenamiento.

---

# Curiosidad

Cuando entrenamos un modelo como ChatGPT, **no se almacenan frases completas**.

Lo que realmente se guarda son cientos de miles de millones de pesos numéricos distribuidos por toda la red.

Por eso un modelo ocupa decenas o cientos de gigabytes.

# Resumen

Has aprendido que:

- Una red neuronal está formada por neuronas artificiales.
- Cada neurona recibe entradas, pesos y un sesgo.
- El conocimiento se almacena en los pesos.
- Las funciones de activación permiten aprender relaciones complejas.
- El cálculo hacia delante se llama Forward Propagation.
- La corrección de errores se realiza mediante Backpropagation.
- Entrenar una red consiste en ajustar millones de pesos para minimizar el error.
