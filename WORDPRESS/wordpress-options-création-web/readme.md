# WordPress pour votre activité de création web

## 1. WordPress.org (auto-hébergé) — Le choix pro

C'est la version **open source**, installée sur votre propre hébergement. C'est de loin la plus adaptée à une activité professionnelle :

- **Contrôle total** sur le code, les plugins, le design
- **Revente de sites** à vos clients sans restrictions
- **Marque blanche** possible (supprimer toute mention WordPress)
- **Hébergement mutualisé, VPS ou cloud** selon vos besoins (OVH, o2switch, Kinsta, WP Engine…)
- Idéal pour livrer des sites clés en main à vos clients

---

## 2. WordPress.com (hébergé) — Plutôt à éviter en pro

La version SaaS gérée par Automattic. Très limitante pour un usage professionnel :

- Plugins tiers bloqués sur les plans bas de gamme
- Vous ne possédez pas vraiment l'hébergement
- Peu adapté à la revente de sites clients

---

## 3. Outils de construction (Page Builders)

Sur WordPress.org, vous avez le choix entre plusieurs approches :

| Outil | Usage |
|---|---|
| **Elementor Pro** | Page builder visuel très répandu, idéal pour les vitrines |
| **Divi** | All-in-one avec thème + builder, bon pour les portfolios |
| **Bricks Builder** | Plus technique, très performant et moderne |
| **Kadence / GeneratePress** | Thèmes légers + blocs Gutenberg natifs |
| **Gutenberg (FSE)** | Éditeur natif WordPress, de plus en plus puissant |

---

## 4. Modèles de travail pour votre activité

### a) Sites livrés et transférés au client
Vous développez sur votre hébergement, puis vous transférez. Utilisez des outils comme **All-in-One WP Migration** ou **WP Staging**.

### b) Maintenance récurrente (MRR)
Vous hébergez les sites de vos clients sous votre compte (revendeur d'hébergement) et facturez une maintenance mensuelle — excellent pour les revenus stables.

### c) Multisite WordPress
Gérez plusieurs sites clients depuis une seule installation WordPress. Pratique mais plus complexe à administrer.

---

## 5. Pour les portfolios spécifiquement

- **Thèmes spécialisés** : Oshine, Bridge, Kalium — très visuels
- **Plugins de galerie** : FooGallery, Modula, Envira Gallery
- **Portfolio avec filtres** : Isotope via Elementor ou JetPortfolio (Crocoblock)

---

## Recommandation pour débuter

Pour une activité de création web, la combinaison la plus efficace est généralement :

> **WordPress.org** + hébergement revendeur (ex: o2switch, LWS) + **Elementor Pro** ou **Bricks** + un thème léger comme **Hello** ou **GeneratePress**
