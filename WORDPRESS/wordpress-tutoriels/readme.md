2. "WordPress pour Débutants : Guide Complet" (sept. 2024)
   Cette vidéo s'adresse aux grands débutants qui souhaitent découvrir ce puissant CMS pour créer un site web.
   👉 Voir sur YouTube YouTube
   [tuto](https://www.youtube.com/watch?v=SE4GTnsEBd8)
3. "Tutoriel Elementor WordPress en Français 2025" – Darrel Wilson
   Un tutoriel axé sur Elementor Pro, avec un hébergement Web comme point de départ. Darrel Wilson est une référence internationale, avec des versions dans plusieurs langues.

   [tuto](https://www.youtube.com/watch?v=4f_aXXnftck)

   Toutes les formations WordPress complètes et gratuites pour apprendre à créer un site professionnel de A à Z.
