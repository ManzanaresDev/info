# Analyse SEO et Performance d'un Site Web

## Performance et Core Web Vitals

### Google PageSpeed Insights
https://pagespeed.web.dev

- Analyse mobile et desktop.
- Mesure les Core Web Vitals (LCP, INP, CLS).
- Donne des recommandations détaillées pour améliorer les performances.
- Gratuit.

### GTmetrix
https://gtmetrix.com

- Analyse approfondie du temps de chargement.
- Waterfall des requêtes réseau.
- Historique des performances.
- Gratuit avec options payantes.

### WebPageTest
https://www.webpagetest.org

- Très complet pour les développeurs.
- Permet de tester depuis différents pays et navigateurs.
- Fournit des captures vidéo du chargement.

---

## SEO

### SEO Site Checkup
https://seositecheckup.com

- Audit SEO complet.
- Vérification des balises meta.
- Sitemap et robots.txt.
- Analyse des liens cassés.

### Ahrefs Webmaster Tools
https://ahrefs.com/webmaster-tools

- Audit technique SEO.
- Détection des erreurs d'indexation.
- Analyse des backlinks.
- Gratuit pour les propriétaires du site.

### Semrush Site Audit
https://www.semrush.com/siteaudit/

- Audit SEO professionnel.
- Vérification des performances techniques.
- Analyse des problèmes critiques et avertissements.

---

## Référencement Google

### Google Search Console
https://search.google.com/search-console/about

Indispensable si le site est déjà en ligne :

- Vérifie l'indexation de Google.
- Détecte les erreurs d'exploration.
- Affiche les requêtes de recherche utilisées pour trouver le site.
- Permet d'envoyer le sitemap.

---

## Accessibilité

### WAVE Accessibility Tool
https://wave.webaim.org

- Détecte les problèmes d'accessibilité.
- Vérifie les contrastes de couleurs.
- Contrôle les balises ARIA et les attributs alt.

---

## Sécurité

### Security Headers
https://securityheaders.com

- Vérifie les en-têtes de sécurité HTTP.
- Note le site de A+ à F.

### SSL Labs Test
https://www.ssllabs.com/ssltest/

- Analyse complète du certificat SSL/TLS.
- Vérifie la configuration HTTPS.

---

## Recommandation pour un projet MERN

Commencer par :

1. Google PageSpeed Insights
2. Google Search Console
3. Security Headers
4. WAVE Accessibility Tool
5. GTmetrix

Ces cinq outils couvrent déjà la majorité des problèmes SEO, performance, accessibilité et sécurité rencontrés sur un site React + Node.js.
