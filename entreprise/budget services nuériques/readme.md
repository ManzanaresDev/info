# Créer mon entreprise de création de sites web en solo avec Incubatest

## Qu'est-ce qu'Incubatest ?

Incubatest est un dispositif de **couveuse d'entreprise** porté par BGE Flandre Création à Dunkerque (33 rue du Ponceau, 59140 Dunkerque). Il vous permet de **tester votre activité avant de vous immatriculer**, dans un cadre légal et sécurisé via un **Contrat d'Appui au Projet d'Entreprise (CAPE)**.

### Ce qu'Incubatest prend en charge pour vous

- ✅ Hébergement juridique (numéro SIRET mis à disposition pour facturer)
- ✅ Appui comptable et administratif
- ✅ Coaching commercial individuel
- ✅ Accès à un CRM
- ✅ Formations collectives au métier de chef d'entreprise
- ✅ Accès au club des entrepreneurs Flandre-Dunkerque
- ✅ Conservation de vos droits sociaux (ARE, RSA…)

> 💡 **Conséquence directe** : en phase de test, vous n'avez **pas besoin** de logiciel de facturation, de CRM, ni de comptabilité — tout est géré par Incubatest.

## Les abonnements logiciels par phase

### Année 1 — Phase Incubatest (~27 €/mois)

L'objectif est de **valider votre marché** en investissant le minimum en logiciels.

| Outil | Prix/mois | Rôle |
|---|---|---|
| Hébergement mutualisé (o2switch ou Infomaniak) | ~8 € | Héberger les sites clients |
| Elementor Pro (ramené/mois, licence annuelle) | ~7 € | Constructeur visuel WordPress |
| Google Workspace | ~6 € | Email pro @votredomaine.fr |
| Figma | Gratuit | Maquettes clients |
| GitHub | Gratuit | Versioning du code |
| **Total** | **~21–27 €** | |

> ⚠️ Pas de logiciel de facturation nécessaire : Incubatest gère la facturation via son SIRET.

---

### Année 2 — Post-immatriculation (~95 €/mois, +252%)

Une fois votre activité validée, vous vous immatriculez (auto-entrepreneur ou SASU). Vous devez désormais gérer vous-même la facturation et la comptabilité, et vos besoins en licences augmentent avec votre portefeuille clients.

| Outil | Prix/mois | Rôle |
|---|---|---|
| Hébergement (plan supérieur) | ~20 € | Plus de sites clients à héberger |
| Figma (plan Pro) | ~15 € | Projets clients avancés |
| CMS & constructeurs (licences multi-sites) | ~22 € | Elementor Pro + thèmes premium |
| Google Workspace | ~14 € | Email + stockage accru |
| Logiciel facturation/compta (Indy ou Freebe) | ~24 € | Obligatoire hors Incubatest |
| **Total** | **~95 €** | |

---

### Année 3 — Phase de croissance (~165 €/mois, +74%)

Votre portefeuille clients s'étoffe, vous affinez votre stack et ajoutez des outils de productivité.

| Outil | Prix/mois | Rôle |
|---|---|---|
| Hébergement (multi-clients) | ~35 € | Montée en puissance |
| Adobe Photoshop ou Figma Pro | ~28 € | Visuels clients complexes |
| CMS, thèmes & plugins premium | ~37 € | Licences multi-sites, WooCommerce… |
| Communication & admin | ~20 € | Google Workspace + Canva Pro |
| Facturation & compta | ~30 € | Plan complet avec gestion TVA |
| **Total** | **~165 €** | |

## Stack recommandé pour démarrer

Pour cibler les **TPE, artisans et commerçants locaux** de la région dunkerquoise, le combo le plus rentable est :

```
WordPress + Elementor Pro + Hébergement o2switch
```

- Le plus demandé par les petites entreprises
- Facile à déléguer/maintenir
- Modèle de revenus récurrents (maintenance mensuelle)

## Conseils pour réussir votre phase Incubatest

1. **Exploitez le réseau** : le club des entrepreneurs Flandre-Dunkerque est votre meilleur levier pour décrocher vos premiers clients.
2. **Minimisez les abonnements** : chaque euro économisé en logiciels est un euro de plus en prospection.
3. **Construisez un portfolio dès le départ** : proposez un tarif réduit à vos 2–3 premiers clients en échange d'un témoignage.
4. **Visez la maintenance récurrente** : un site livré + un contrat de maintenance à 50–100 €/mois = revenus stables.
5. **Documentez votre activité** pour faciliter l'immatriculation à l'issue de la période de test.
