# Forfait annuel de maintenance et d'hébergement d'un site web

Ce document présente le forfait annuel permettant d'assurer la
disponibilité, la sécurité et l'évolution d'un site web professionnel.

------------------------------------------------------------------------

# Services inclus

## Hébergement sécurisé

Le site est hébergé sur une infrastructure professionnelle garantissant
:

-   haute disponibilité du site
-   rapidité d'accès
-   protection contre les interruptions de service
-   surveillance du serveur

------------------------------------------------------------------------

## Nom de domaine

Gestion complète du nom de domaine :

-   renouvellement annuel
-   configuration DNS
-   gestion technique

Exemples : - monsite.fr - monsite.com

------------------------------------------------------------------------

## Certificat de sécurité SSL

Installation et renouvellement du certificat SSL permettant :

-   une navigation sécurisée (HTTPS)
-   l'affichage du cadenas de sécurité dans le navigateur
-   la protection des données des utilisateurs

------------------------------------------------------------------------

## Sauvegardes régulières

Des sauvegardes du site et de la base de données sont réalisées afin de
pouvoir :

-   restaurer le site en cas de problème
-   récupérer les données en cas d'erreur
-   protéger le contenu contre toute perte accidentelle

------------------------------------------------------------------------

## Maintenance technique

La maintenance comprend :

-   surveillance du bon fonctionnement du site
-   mises à jour techniques
-   correction d'éventuels bugs
-   assistance technique

------------------------------------------------------------------------

## Optimisation SEO de base

Mise en place des éléments nécessaires au référencement naturel :

-   optimisation des balises principales
-   configuration SEO du site
-   suivi technique du référencement
